# Prompt Test Copy Button and Prompt Notice Update

## Changes

- Added a warning notice in the Data Engineering View detail panel explaining that the displayed prompts are conceptual prototype ideas, not exact validated production prompts.
- Changed Prompt Test accordion labels from `Prompt 1` style to `1. Prompt` style.
- Added a copy control next to each prompt in the Prompt Test page.

## Rationale

The Prompt Test page is intended for presentation and methodological explanation. The prompts demonstrate an auditable prompt-chain concept for the ABF Annual Report 2025 sample, but they should not be presented as final scientifically validated prompts without further testing.
