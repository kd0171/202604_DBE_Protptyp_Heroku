# ABF re-extraction and source-link update

This version incorporates uploaded ABF Annual Reports 2021–2025 into the `documents/04_abf-sugar/` folder and regenerates dashboard CSVs for ABF Sugar.

## Main data updates

- ABF Sugar financial time series expanded from 2021 to 2025.
- ABF Sugar operations profile updated from the 2025 Annual Report: 28,000 people, 16 plants, eight countries and approximately 4 million tonnes of annual sugar capacity.
- ABF Sugar risk and investment events added for Vivergo closure, Azucarera restructuring/impairments and Ubombo growth investment.
- Source columns now include `source_url`, `source_page` and `source_link` where available. The app displays `source_link` in evidence tables so source documents are shown as PDF links rather than local file paths.
- Interactive Mode questions were rewritten to be more specific and business-oriented.

## Interpretation note

ABF Sugar is a peer competitor, not the focal company. Nordzucker remains the focal company. ABF Sugar data is useful as a peer benchmark because it shows how a low European sugar-price environment can move from profitability pressure to restructuring and impairment events.
