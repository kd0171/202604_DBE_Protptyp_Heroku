# Tutorial text and media size update

This update restructures the Data Engineering tutorial so each step is easier to follow.

## Changes

- The first tutorial step is now a start guide rather than an overview.
- The first step no longer displays an empty media placeholder.
- Each tutorial step is written with short headings and concise explanations.
- GIF-based tutorial steps are displayed larger by using an extra-large modal and a larger media frame.
- The Prompt/Output tutorial step explicitly explains that the red boxes in the GIF highlight the Prompt and Output areas.
