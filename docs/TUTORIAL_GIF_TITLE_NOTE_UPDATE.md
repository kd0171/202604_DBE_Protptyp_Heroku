# Tutorial GIF title note update

This update removes file-location reminder messages from tutorial GIF hint boxes and missing-media placeholders.

Instead, tutorial GIF areas now display a short note that the title shown inside a GIF may reflect an earlier version of the prototype. This avoids confusing deployment users with internal asset paths while still explaining why a recorded GIF may not exactly match the current app title.

Affected areas:

- Home tutorial
- Prompt Test tutorial
- Data Analysis tutorial
- Data Engineering tutorial media display note
